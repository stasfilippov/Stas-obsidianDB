## Переход с npm на yarn 
1. если мы добавим что-то с помощью **yarn** 
2. появятся оба файла .lock
3. удаляем yarn.lock
4. удаляем package-lock.json
5. удаляем node_modules (из проводника)
6. из терминала - yarn install
